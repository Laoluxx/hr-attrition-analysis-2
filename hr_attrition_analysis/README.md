# IBM HR Attrition Analysis

## Project Overview
This project analyzes employee attrition data to understand patterns behind why employees leave the company.

## Dataset
IBM HR Analytics Employee Attrition dataset (CSV).

## Objectives
- Calculate overall attrition rate
- Identify departments with the highest attrition
- Compare age and income patterns between employees who left vs stayed

## Tools Used
- Python
- Pandas

## Key Insights
- Attrition Rate: 16.2%
- Department with highest attrition: Research & Development
- Average age of employees who left: 33.6 years
- Average monthly income of employees who left: $4787.09

## How to Run
1. Install dependencies:
   - `pip install pandas`
2. Run the script:
   - `python employee_attrition_analysis.py`

## Files
- `employee_attrition_analysis.py` — main analysis script
